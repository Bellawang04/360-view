# 用户使用路径

> 演示时走通3条故事，每条不超过10分钟

---

## 路径一：销售视角

**角色**：客户经理  
**起点**：Fiori Launchpad → My Sales Overview (F2200)

```
1. 登录 Fiori Launchpad
   → 看到 KPI Tile：高流失风险 12 / 合同即将到期 7 / 长期未下单 5

2. 点击「高流失风险」Tile
   → 跳转到客户列表，筛选条件：流失风险=高

3. 点击某客户（如：苏州博众精工）
   → 进入 Customer 360° View（F2187A）

4. 在360页面依次展示：
   ├── 头部徽章：财务风险=高 / 合规状态=注意 / RFM=D级
   ├── 卡片A 财务风险雷达：AR逾期90天+ ¥0.5万（红色）
   ├── 卡片B 销售活跃度：距上次下单 63天（红色预警）
   ├── 卡片C 合规状态：年度框架合同剩余28天（橙色）
   └── 卡片E 外部风险：工商正常 / 无司法记录（演示数据）
```

**验收点**：能看到逾期金额、最近下单时间、RFM等级、外部风险状态

---

## 路径二：财务视角

**角色**：财务专员  
**起点**：Fiori Launchpad → Order to Cash Dashboard (F4663)

```
1. 登录 Fiori Launchpad（财务角色）
   → 看到 KPI Tile：AR逾期30天+ ¥2,340万（涉及18客户）/ 信贷超限 4 客户

2. 点击「AR逾期」Tile
   → 跳转客户列表，按逾期金额降序排列

3. 点击金额最大的客户
   → 进入 Customer 360° View

4. 展示财务风险雷达卡片：
   ├── 30/60/90天各档逾期金额可视化
   ├── 信贷占用率进度条
   └── 展开详情 → 跳转 FBL5N 查完整明细
```

**验收点**：财务工作台能看到信用超限清单，按逾期金额排序，点击进详情

---

## 路径三：管理层视角

**角色**：销售总监  
**起点**：Fiori Launchpad → Sales Management Overview (F2601)

```
1. 登录 Fiori Launchpad（管理层角色）
   → 看到客户价值分层：A 34% / B 28% / C 23% / D 15%
   → 看到风险概览：高流失风险 12 / 本月新增预警 27

2. 点击「D级客户」色块
   → 跳转客户列表，筛选 RFM=D

3. 在列表页展示：
   ├── 每行显示：客户名 / 销售额 / AR状态 / RFM / 风险状态
   └── 点击「导出」→ 导出D级客户名单Excel

4. 管理层追问某客户为什么是D级
   → 点击进入360页面，展示RFM各维度分数
```

**验收点**：能看到A/B/C/D分布，能筛选D级客户，能导出名单
